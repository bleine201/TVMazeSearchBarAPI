## Installation

### Git clone

```bash
git clone git@github.com:bleine201/TVMazeSearchBarAPI.git
```

### Running App

```bash
npm install
npm start
```

### If any error when running

Create an environmment file

```bash
touch .env
```

Then copy/paste

```bash
SKIP_PREFLIGHT_CHECK=true
```
